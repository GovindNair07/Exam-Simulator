# 📝 Exam Simulator

## 📌 Overview
Exam Simulator is a Java-based application developed using the Spring Framework (XML configuration) and Hibernate ORM. The system simulates a real-time exam environment where students can take tests, and administrators can manage questions and users.

The project demonstrates backend development, database integration, and interactive UI design using Java Swing and multimedia components.

---

## 🚀 Features
- 👤 Student Registration & Login  
- 🧑‍💼 Admin Dashboard for managing exams and users  
- 📝 Interactive exam simulation system  
- 📊 Performance evaluation and result handling  
- 💾 Database integration using Hibernate  
- 🎬 Multimedia support (GIF animations, audio feedback)  
- 🖥️ GUI-based interface using Java Swing  

---

## 🛠️ Tech Stack
- **Language:** Java  
- **Framework:** Spring Framework (XML-based configuration)  
- **ORM:** Hibernate (`.hbm.xml` mapping)  
- **Database:** MySQL  
- **UI:** Java Swing  
- **Build Tool:** Apache Ant  
- **IDE:** NetBeans  

---

## 🏗️ Architecture
The project follows a layered architecture:

- **Presentation Layer:** Java Swing (GUI forms)  
- **Controller/Service Layer:** Spring-managed classes  
- **Data Access Layer:** Hibernate DAO classes  
- **Database Layer:** MySQL  

---

## 📂 Project Structure
- `/src` → Java source code  
- `/src/spring` → Spring configuration and DAO layer  
- `/src/spring/hib` → Hibernate entities and mappings  
- `/nbproject` → NetBeans project configuration  
- `build.xml` → Ant build file  

---

## ⚙️ How to Run
1. Open the project in NetBeans  
2. Configure MySQL database  
3. Update database credentials in configuration files  
4. Build project using Apache Ant  
5. Run the application
